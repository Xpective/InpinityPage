How To / FAQ V6 Updates

A quick guide to how the main gameplay loops work with V6 contracts.

What's new in V6?

V6 brings several improvements:

  • Enhanced attack preview with exact steal amount
  • Pirate and Farming boosts with clear expiry
  • Improved subgraph performance with farmV6S/attackV6S
  • More accurate resource claiming previews
How does minting work?

Choose coordinates, select a payment option, and mint a Pyramid Block NFT. After minting, the block can later be revealed and used for farming, defense, and attacks.

What does revealing a block do?

Revealing unlocks the block's hidden state and exposes its rarity-related attributes. That makes the block fully usable in the wider game flow.

How do V6 attacks work differently?

V6 attacks now show you the exact steal amount before you commit. You also have better control over which block attacks from, and can preview all resource lanes to find the most valuable target.

What are boosts and how do I use them?

Farming Boost: Increases resource production by 2x for a selected duration.
Pirate Boost: Enhances attack effectiveness and reduces travel time.
Look for the ⚡ buttons in each section to activate boosts.

What is Pitrone Exchange used for?

The exchange area allows you to monitor the current INPI ↔ PIT relation and swap between both sides according to the available pool state and configured logic.
